# Examples

These examples show how to start and steer the swarm without triggering any
release action. They mirror `examples/prompts.md` in the repository.

- [Kickoff prompts](kickoff-prompts.md) — a universal starter, an execute-approved
  -plan prompt, and an independent-review prompt.

Keep the [operating model](../organization/operating-model.md) in mind: start in
Discover and Plan modes, pick the smallest useful team, and stop before any
release action until a human approves.
