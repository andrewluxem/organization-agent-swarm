# Decision rights

Every decision has one accountable owner. This mirrors the decision-rights table
in `team/operating-model.md`.

| Decision | Owner |
|---|---|
| Task routing | Worldwide orchestrator |
| Division plan | Division lead |
| Specialist implementation | Assigned specialist |
| Measurement design | Consumer Marketing Analytics |
| Independent acceptance review | Cross-Channel Measurement and Optimization |
| Publish, deploy, send, spend, merge, push, or production change | Human |

## Independent review

The independent reviewer — Cross-Channel Measurement and Optimization — evaluates
whether the evidence supports a launch, continue, modify, or stop decision. It
must **not** edit the implementation it reviews. Keeping authoring and review
separate is what keeps acceptance honest.

## Human release authority

No agent holds release authority. Publishing, deploying, sending, spending,
merging, pushing, and changing production data or configuration are reserved for
a human. See [release gates](../governance/release-gates.md) in the governance section for the
full list of gated actions.
